✅ CÓMO USAR
═════════════════════════════════════════════════════════════════════════════

1. CONFIGURAR API KEY (una sola vez)
   ──────────────────────────────────
   a) Ve a: https://aistudio.google.com/app/apikey
   b) Crea una API key (es gratis y rápido)
   c) En PowerShell, ejecuta:
      $env:GEMINI_API_KEY = "tu_api_key_aqui"

2. EJECUTAR EL PROYECTO
   ───────────────────────
   $ python manage.py runserver

3. USAR LA FUNCIONALIDAD
   ──────────────────────
   a) Ve a http://localhost:8000/productos/
   b) Selecciona ingredientes
   c) Haz clic en "🤖 Receta con IA"
   d) ¡Disfruta tu receta generada!
